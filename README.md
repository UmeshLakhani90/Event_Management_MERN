# Event Management MERN Application

## 📌 Description
A web application to manage events and user registrations.

## 🚀 Tech Stack
- MongoDB
- Express.js
- React
- Node.js

## ✨ Features
- User authentication
- Create & manage events
- Book event tickets
- Admin dashboard

## ⚙ Installation
1. Clone the repository
2. Run: npm install
3. Run: npm start
